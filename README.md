# Portfolio — Designer & UI/UX Expert Website

A premium, modern portfolio website tailored for Designers and UI/UX Experts. This project features a sophisticated minimalist design style utilizing a rich, custom typography pairing (`Playfair Display` and `DM Sans`) along with an elegant color palette spanning warm creams, high-contrast jets, and warm accent elements.

## ✨ Features

- **Grainy Overlay & Glassmorphism Navigation:** Fixed navigation bar built with backdrop filtering on top of an elegant, custom SVG-generated inline procedural noise texture applied across the body.
- **Dynamic CSS Animations:** Custom `@keyframes` fade-up staging for fluid component introduction on hero element rendering.
- **Infinite Micro-Interactions:** Custom loop-marquee tracking bar for highlighting dynamic text tickers or technical expertise highlights.
- **Comprehensive Portfolio Grids:** Structured grid template configuration with variable layout spans matching different showcase requirements (Hero, Narrow, Wide).
- **Client Testimonials Carousel & Metric Trackers:** Visually engaging callout sections detailing dynamic project statistics and sleek blockquotes.
- **Responsive Navigation Mechanics:** Interactive slide-out mobile drawer utility complete with animated pure-CSS hamburger transformation triggers.
- **Functional Interactive Contact Form:** Floating label inputs paired with instant post-submission visual success states built on underlying DOM handling.

## 🛠️ Tech Stack

- **Structure:** Semantic HTML5 Markup
- **Styling:** Modern CSS3 featuring Native CSS Custom Variables (Design Tokens), Flexbox layout systems, CSS Grid layout layers (`grid-template-columns: repeat(12, 1fr)`), clamp-based typography sizing (`clamp()`), and media viewport breakpoints.
- **Fonts:** Imported typography layers via Google Fonts (`Playfair Display`, `DM Sans`).
- **Interactions:** Lightweight vanilla ECMAScript JavaScript.
